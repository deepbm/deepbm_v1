Shop

  1. Introduction

    쇼핑몰 메인 웹 페이지를 제작했습니다.

  2. Stack
    • html5
    • css3
    • bootstrap
    • javascript
    • jquery
    • git
    • github
  3. What did I do
    • html5 표준에 추가된 시맨틱 태그 사용하여 웹 사이트 구축
    • bootstrap 프레임워크를 사용하여 스타일 지정
    • 반응형으로 웹 페이지 제작
    • fontawesome 아이콘 적용
    • sign in 버튼을 클릭하면 화면 전체에 로그인 모달창이 뜨는 기능 구현
    • 로그인 모달창의 닫기 버튼에 마우스 커서를 올리면 회전하는 애니메이션 기능 구현
    • 로그인 입력창이 빈 값인 상태에서 로그인 버튼을 클릭하면 경고창이 뜨는 기능 구현
    • 스크롤하여 특정 위치에 도달할 시 top 버튼이 생성되고, 해당 버튼을 누르면 화면 최상단으로 이동하는 기능 구현
    • Ajax를 활용하여 상품 더보기 버튼을 클릭하면 서버에서 데이터를 가져와 화면에 적용되는 기능 구현

  1. Introduction

    입력한 점심 메뉴 중 하나를 선택해주는 프로그램을 제작했습니다.

  2. Stack
    • html5
    • css3
    • javascript
    • jquery
    • git
    • github
  3. What did I do
    • 구글 웹 폰트 적용
    • to do list 코드를 재사용함으로써 코딩 시간 단축
    • 재사용 가능하게 클래스를 지정해 사용함으로써 코드의 가독성을 높임
    • fontawesome 아이콘 적용
    • 리스트에 있는 삭제 버튼을 누르면 해당 리스트가 삭제되는 기능 구현

    • 자체 데이터에서 랜덤으로 하나 선택되게 하는 기능 구현
    • 엔터키를 누르거나 ADD 버튼을 누르면 리스트에 등록되는 기능 구현
    • 배열에 데이터를 저장하여 기능 조작에 따른 데이터 처리(create, read, remove)
